# 阅前必看
1. 手册说明的步骤均为具体方块的坐标，请知道如何使用F3调试页面右下角的`target block`**定位方块**，注意，这里指的不是左上角的玩家坐标
2. 除非有特殊说明，**请不要变动**步骤中未提及的方块与实体，包括放置、破坏、容器取出、容器放入、开关门，除非你对这台机器的原理非常了解
3. 尽量不要只拿取不刷取，保证物资平衡
4. 在不确定时，向服务器内或身边的人**求助**也是一个不错的选择

- **维度构型：** 单维度

**使用说明**

## 高速熔炉组
一个建立在北海的320核心高速熔炉组
- **产出：** 任意盒装熔炼配方输出
- **输入：** 任意盒装熔炼配方输入
- **坐标：** 
- **挂机构型：** 主动式
- **维度构型：** 单维度

**使用说明**
- 1. 站在`()@主世界`的`橡木活版门`上
- 2. 将盒装物品放入`()@主世界`的`木桶`中
- 3. 检查无误后拉下`()@主世界`的`拉杆`，等待散装产物输出
- 4. 将产物转移至`()@主世界`的`潜影盒`中，满盒后将自动送入潜影盒分类机，目前有石头、平滑石头、玻璃、其他四种分类
- 5. 产物停止输出后，拉回`()@主世界`的`拉杆`
- 在`()@主世界`的`箱子`中拿取盒装熔炼物品

**风险提示**
- `🟥损坏`：在机器运行中离开将导致燃料部分损坏，可以启动位于`()@主世界`的区块加载器
- `🟧失能`：投入任何不在熔炼配方输入的物品会导致一个或多个熔炉堵塞，需要检查320个熔炉和320个漏斗
- `🟧失能`：在机器运行中进入机器内部会导致物流矿车时序错位，需要重新填装矿车
- `🟨物资损失`：在机器投料口输入散装物资将直接被岩浆烧毁

## 模板
设备概述
- **产出：** **物资A(10k/h)**、物资B(-物资C)
    - **加粗**代表为此种物资的最优产出点
    - (-物资)代表机器对此种产物存在二次合成打包路线，如金粒(-金锭-金块)代表机器可以将金粒合成金锭，金锭合成金块。
- **输入：** 物资C
- **坐标：** (x,y,z)@维度
    - 如无补充说明，坐标为挂机点
- **挂机构型：** 无/被动式/长按式/连点式/主动式
    - 无：玩家无需挂机
    - 被动式：玩家在挂机点上不需要进行任何操作
    - 长按式：玩家在挂机点上需要长按鼠标按键进行操作(参考F3+T的长按保持)
    - 连点式：玩家在挂机点上需要快速点击鼠标按键进行操作(参考连点器的使用)
    - 主动式：玩家在挂机点上需要主动进行操作
- **维度构型：** 单维度/半双维度/双维度(被动式挂机玩家ID)
    - 单维度：怪物的生成、运输、处死均在同一维度，任何情况下仅需一名玩家挂机
    - 半双维度：怪物的生成、处死在同一维度，运输在另一维度，当运输维度存在玩家时，需要一名玩家被动式挂机在运输维度
    - 全双维度：怪物的生成、处死在不同维度，任何情况下均需要一名玩家被动式挂机在生成维度，主玩家则在处死维度
    - 部分(半)双维度已经准备好了被动式挂机玩家，会在括号内注明，直接登录即可
使用说明
